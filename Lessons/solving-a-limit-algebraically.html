Solving a Limit Algebraically

Now that you understand what a limit is, it's time to learn how to use algebra to solve a limit. Let's take the graph we discussed in the first lesson: $$f(x)=\frac{x^3+1}{x+1}$$ Earlier we mentioned that the domain of this function is \(x\ne -1\). This is because at \(x=-1\), the denominator is zero.

But something is different this time. Not only is the denominator 0, but the numerator is 0, too: $$f(-1)=\frac{(-1)^3+1}{(-1)+1}=\frac{0}{0}=\textrm{indeterminate}$$ When the value of a function is \(\frac{0}{0}\), the numbers almost always have a way of "canceling each other out."
The function seems to approach the point \((-1,3)\). And now we will show you how to remove the zero: factoring it out. In the numerator, \(x^3+1\) can be factored as shown: $$\frac{x^3+1}{x+1}=\frac{(x^2-x+1)*(x+1)}{x+1}$$ The reason the function equals \(\frac{0}{0}\) at -1 is because \(x-1=0\), so now we will cancel it out to get the following function: $$g(x)=x^2-x+1$$ The graph of this function is shown below: It looks almost identical to the first function, but this time the function has no hole at -1. It has the point (-1,3).

Plugging -1 into the new function, we get $$g(-1)=(-1)^2-(-1)+1 = 3$$ And that's it!
