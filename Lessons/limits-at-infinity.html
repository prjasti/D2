Limits at Infinity

There comes a point in our lives when someone finally tells us the truth about life: infinity is not actually a number. When we look at a graph, we can try as hard as we can to reach infinity but all we do is approach it. Do you see where I'm going with this?

And that's why we use limits to find values of functions at infinity.

Let's start with the following function: $$f(x)=\frac{1}{x}$$ We can usually look at a graph and predict what it's end behavior is. We see in this example that \(\frac{1}{x}\) keeps getting smaller, getting infinitesimaly close to 0 without reaching it. Try tracing the graph above and see for yourself if you don't believe me.

This limit can be written as $$\lim_{x\to \infty} \frac{1}{x} = 0$$ Now for the algebra behind this.
